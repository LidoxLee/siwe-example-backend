# Sign in with Ethereum simple example - using siwe-go

It's very simple parse siwe message, and verify address example.

## How to use?

### 1. Clone this repo.

### 2. Install go package

```
go mod tidy
```

### 3. Run http server at local

```
go run main.go
```
